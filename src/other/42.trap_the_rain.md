## 42. 接雨水

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

![](https://assets.leetcode-cn.com/aliyun-lc-upload/uploads/2018/10/22/rainwatertrap.png)

示例:
```
输入: [0,1,0,2,1,0,1,3,2,1,2,1]
输出: 6
```

--- 

### golang 解法

#### 暴解

- 时间复杂度：O(n²）
- 空间复杂度：O(1)

```golang
// 暴力解法
func trap(height []int) int {
	var result int
	length := len(height)
	min := func(i, j int) int {
		if i < j {
			return i
		}
		return j
	}

	for i := 1; i < length-1; i++ {
		var leftMax int
		for j := i - 1; j >= 0; j-- {
			if height[j] > leftMax {
				leftMax = height[j]
			}
		}

		var rightMax int
		for k := i + 1; k < length; k++ {
			if height[k] > rightMax {
				rightMax = height[k]
			}
		}

		columnResult := min(leftMax, rightMax) - height[i]
		if columnResult > 0 {
			result += columnResult
		}
	}

	return result
}
```

#### 动态规划

- 时间复杂度：O(n)
- 空间复杂度：O(n)

```golang
// 动态规划
func trap(height []int) int {
	length := len(height)
	if length == 0 {
		return 0
	}

	max := func(i, j int) int {
		if i > j {
			return i
		}
		return j
	}
	min := func(i, j int) int {
		if i < j {
			return i
		}
		return j
	}

	head := make([]int, length)
	head[0] = height[0]
	for i := 1; i < length; i++ {
		head[i] = max(head[i-1], height[i])
	}

	tail := make([]int, length)
	tail[length-1] = height[length-1 ]
	for j := 1; j < length; j++ {
		tail[length-1-j] = max(tail[length-j], height[length-1-j])
	}

	var resp int
	for k := 0; k < length; k++ {
		resp += min(head[k], tail[k]) - height[k]
	}
	return resp
}
```

#### “挤水法”

容器先填充“水”，再把会流走的水挤掉。

- 时间复杂度：O(n)
- 空间复杂度：O(n)

```golang
// 挤水
func trap(height []int) int {
	var maxHeight, index int
	length := len(height)

	// 求出数组中最大(高)
	for i := 0; i < length; i++ {
		if height[i] > maxHeight {
			maxHeight = height[i]
			index = i
		}
	}

	capacity := make([]int, length)
	for j := 0; j < length; j++ {
		capacity[j] = maxHeight - height[j]
	}

	var sub int
	for i := 0; i < index; i++ {
		if height[i] > sub {
			sub = height[i]
			capacity[i] = 0
		} else {
			capacity[i] = sub - height[i]
		}
	}

	sub = 0
	for j := length - 1; j > index; j-- {
		if height[j] > sub {
			sub = height[j]
			capacity[j] = 0
		} else {
			capacity[j] = sub - height[j]
		}
	}

	var result int
	for _, v := range capacity {
		result += v
	}
	return result
}
```

#### “倒水法”

找出最高点，在它“头上”倒水，水会往两边走并把“坑”填满。

- 时间复杂度：O(n)
- 空间复杂度：O(1)

```golang
// 倒水
func trap(height []int) int {
    if len(height) <= 1 {
        return 0
    }
    
	var maxHeight, index int
	length := len(height)

	// 求出数组中最大(高)
	for i := 0; i < length; i++ {
		if height[i] > maxHeight {
			maxHeight = height[i]
			index = i
		}
	}

	var result int
	head := height[0]
	for i := 0; i < index; i++ {
		if height[i] > head {
			head = height[i]
		} else {
			result += head - height[i]
		}
	}

	tail := height[length-1]
	for j := length - 1;j >index;j-- {
		if height[j] > tail {
			tail = height[j]
		} else {
			result += tail - height[j]
		}
	}
	
	return result
}
```
